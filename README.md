# Education Platform

Платформа для управления учебными курсами и оценивания работ студентов. Преподаватели могут создавать курсы, организовывать материалы по модулям и урокам, выставлять домашние задания и проверять работы студентов.

## Проект

**Education Platform** — это веб-приложение для онлайн-обучения, ориентированное на преподавателей. Система позволяет:
- Управлять структурой курсов (модули → уроки → материалы)
- Создавать различные типы уроков: видеоуроки, статьи, тесты и домашние задания
- Прикреплять файлы и ссылки к урокам
- Просматривать работы студентов и выставлять оценки с комментариями

## Основные сущности

- **Teacher** – преподаватель, создатель курсов
- **Student** – студент, проходящий курсы
- **Course** – учебный курс
- **Module** – раздел курса (группирует уроки)
- **Lesson** – занятие (видео, статья, тест или задание)
- **LessonMaterial** – учебный материал к уроку (файл или ссылка)
- **Assignment** – домашнее задание
- **AssignmentSubmission** – работа студента с оценкой и комментариями
- **Test** – тестовое задание с вопросами

## API основные эндпоинты

- `GET /api/me` – информация о текущем преподавателе
- `GET /api/courses` – список всех курсов
- `GET /api/teachers/:teacherId/courses` – курсы преподавателя
- `GET /api/courses/:id` – курс с полной структурой (модули, уроки, материалы)
- `PATCH /api/submissions/:submissionId/grade` – выставление оценки за работу

## Запуск проекта

```bash
# Клонировать репозиторий
git clone https://github.com/skufirovan/education-platform.git
cd education-platform

# Запустить контейнеры
docker compose up --build

# В новом окне терминала - заполнить БД тестовыми данными
docker compose exec backend npm run seed

# Открыть приложение
# http://localhost:5173/courses
```
